# Hardware-Software Data Acquisition System for Measuring the Grip Strength in Children

This repository contains a data acquisition system with:
- Mobile interface developed in MIT App Inventor
- Desktop virtual instrument developed in LabVIEW
- Two DAQ boards:
  - PIC-based microcontroller board
  - ESP32 NodeMCU-based board

Both interfaces can operate with either DAQ board.

This project is a data acquisition system for measuring grip strength in children. It consists of two data acquisition boards, one based on the PIC18F2550 microcontroller and the other on the NODE MCUESP32S development board. It features two types of graphical interfaces, one implemented with LabVIEW and the other designed with App Inventor.The grip strength of the index, middle, ring, and pinky fingers is measured using FSR-402 pressure sensors
